# Карта Игр Steam

Этот проект создает интерактивную 2D-карту игр из Steam, где положение каждой игры определяется на основе ее тегов.

## Как запустить проект

### 1. Установка зависимостей

Перед первым запуском убедитесь, что вы находитесь в вашем активированном виртуальном окружении (`.venv`), и установите все необходимые библиотеки:

```bash
pip install -r requirements.txt
```

### 2. Запуск API-сервера

Этот сервер предоставляет данные для веб-интерфейса. Запустите его в **первом терминале**:

```bash
python -m uvicorn app:app --reload
```

### 3. Запуск веб-интерфейса

Это приложение отображает интерактивную карту в вашем браузере. Запустите его во **втором терминале**:

```bash
python -m streamlit run main.py
```

После выполнения второй команды в вашем браузере автоматически откроется новая вкладка с картой.
